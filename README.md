
# 🚀 Proyecto Marketplace - Autenticación

## 📌 Descripción
Este proyecto implementa un módulo de autenticación para un marketplace digital usando Node.js y Express.

## ⚙️ Tecnologías
- Node.js
- Express
- JWT
- HTML, CSS, JavaScript

## 🔐 Funcionalidades
- Login de usuarios
- Registro de usuarios
- Generación de token JWT
- Validación con middleware

## 🧱 Arquitectura
- Controllers
- Services
- Routes
- Middlewares

## 🌐 Frontend
Ubicado en la carpeta `/frontend`:
- index.html
- login.html
- registro.html

## 🚀 Ejecución

```bash
node app.js
